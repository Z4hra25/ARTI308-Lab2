# ARTI308-Lab2
Machine Learning Lab 2

## Dataset Description
The dataset used in this lab is the Iris dataset from the UCI Machine Learning Repository.
It contains measurements of iris flowers and is suitable for a machine learning classification task.

## Machine Learning Problem
The goal of this project is to predict the species of an iris flower based on its physical measurements.
This is a supervised multi-class classification problem.

- Problem Type: Classification  
- Target Variable: species
